## Objetivo

Realizar estudo e levantamento de tecnologias que podem ser usadas no projeto, com foco em áreas específicas de desenvolvimento, para iniciar a implementação de maneira consistente, e, ao final, definir quais serão utilizadas.

## Áreas de desenvolvimento

- **Frontend:** Responsável pela interface que comunica com o usuário, além de enviar requisições para o Backend;
- **Backend:** Responsável pela lógica e regras do sistema, além de disponibilizar a **API** para a comunicação com Frontend;
- **Banco de dados:** Responsável por armazenar e organizar as informações da aplicação;
- **Infraestrutura:** Responsável por garantir que o sistema possa ser executado de maneira estável, funcionando em todos os computadores;
- **Testes:** Executa testes para validar o funcionamento da aplicação.

## Alternativas de tecnologias e linguagens

### Frontend

- **Linguagens:** JavaScript; TypeScript;
- **Frameworks:** React; Angular; Larave; Vue.jsl;

### Backend

- **Linguagens:** Python; C#; JavaScript;
- **Frameworks:** Node.js; SpringBoot; Node.js;
- **FastAPI**

### Banco de dados

- **Sistemas:** PostgreSQL; MySQL; MongoDB;

### Infraestrutura

- **Ferramentas:** Kubernetes; Docker; GitHub Actions (CI/CD);

### Testes

- **Ferramentas:** Jest; Cypress; Selenium; JUnit; Pytest;
