📐 Como funciona a análise
O sistema extrai os discursos mais recentes do parlamentar e os compara, usando similaridade, com as ementas das votações registradas. Votações com ≥ 70% de afinidade com o discurso de referência são marcadas como ✅ Coerentes. Isso indica que o parlamentar votou sobre temas compatíveis com o que discursou publicamente.
